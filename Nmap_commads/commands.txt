ðŸ”¹ Host Discovery (Find Live Hosts)
Description
Used to identify which devices are active (host is up) in a network without scanning ports.

Commands

nmap -Pn 192.168.1.10

ðŸ”¹ Port Scanning
Description
Detects open, closed, and filtered ports on a target system.

Commands

sudo nmap -sS 192.168.1.10
nmap -sT 192.168.1.10
sudo nmap -sU 192.168.1.10
nmap -p 22,80,443 192.168.1.10
nmap -p- 192.168.1.10

ðŸ”¹ Service & OS Detection
Description
Identifies running services, their versions, and the operating system of the target.

Commands

nmap -sV 192.168.1.10
sudo nmap -O 192.168.1.10
sudo nmap -A 192.168.1.10

ðŸ”¹ Timing & Speed Control
Description
Controls how fast or slow Nmap sends packets to balance stealth and speed.

Commands

nmap -T3 192.168.1.10
nmap -T4 192.168.1.10

ðŸ”¹ Firewall & IDS Evasion
Description
Techniques used to bypass firewalls and intrusion detection systems.

Commands

ðŸ”¹ Nmap NSE Scripts
Description
Nmap Scripting Engine (NSE) automates vulnerability checks and service enumeration.

Commands

nmap -sC 192.168.1.10
nmap --script vuln 192.168.1.10
nmap --script smb-vuln* 192.168.1.10
nmap --script ftp-anon 192.168.1.10

ðŸ”¹ Output & Reporting
Description
